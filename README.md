# 📦 Controle de Estoque

Sistema simples de controle de estoque desenvolvido com React e Google Sheets como banco de dados.

## 🚀 Tecnologias utilizadas

- React
- JavaScript
- Google Sheets (como banco de dados)
- Google Apps Script (API)

## ⚙️ Funcionalidades

- ✅ Adicionar produtos
- ✅ Listar produtos
- ✅ Excluir produtos

## 🧠 Como funciona

O sistema utiliza o Google Sheets como banco de dados, acessado através de uma API criada com Google Apps Script.

```bash
npm install
npm run dev
